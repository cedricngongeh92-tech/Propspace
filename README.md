# PropSpace

PropSpace is an exam project using a full-stack JavaScript setup.

## Stack

- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB

## Setup

Install frontend dependencies:

```bash
cd client
npm install
```

Install backend dependencies:

```bash
cd server
npm install
```

Create a backend environment file:

```bash
cd server
copy .env.example .env
```

Run the frontend:

```bash
cd client
npm run dev
```

Run the backend:

```bash
cd server
npm run dev
```
